🔍 About This App

This Churn Prediction App is an end-to-end machine learning application built using FastAPI. It enables businesses to predict whether a customer is likely to churn based on key features.

📊 Machine Learning Pipeline

  • 📥 Data Ingestion: Load raw CSV data and split into train-test sets.
  • 🧪 Data Validation: Check for missing values and required schema.
  • 🔄 Data Transformation: Apply encoding, scaling, and feature engineering.
  • 🤖 Model Training: Use models like XGBoost, Random Forest, and Logistic Regression.
  • ⚙️ Threshold Tuning: Optimize the decision threshold for better recall.
  • 📈 Evaluation: Use precision-recall tradeoff and F1-score for robust evaluation.

🧪 How to Use

  • 🔹 Single Prediction: Enter customer details in the form and view real-time prediction and probability.
  • 🔹 Batch Prediction: Upload a CSV file of multiple customers and download the prediction results in a CSV file.

✅ Whether you're evaluating individual customers or analyzing bulk data, this app offers speed, transparency, and accuracy.
